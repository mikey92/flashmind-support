자주 묻는 질문(FAQ)

데이터는 어디에 저장되나요?

모든 데이터는 Firebase에 안전하게 저장되며, 기기 간 자동 동기화됩니다.

오프라인에서도 사용할 수 있나요?

네, 한 번 다운로드한 데이터는 오프라인에서도 사용할 수 있습니다.

광고를 제거하고 싶어요.

앱 내 설정에서 프리미엄 구독을 통해 광고를 제거할 수 있습니다.

계정을 삭제하려면?

앱 내 설정 > 계정 관리에서 계정 삭제를 요청할 수 있습니다.

개인정보처리방침

1. 수집하는 개인정보

  • 회원가입 시: 이메일 주소, 닉네임
  • 자동 수집: 디바이스 UUID, 플랫폼 정보(iOS), 앱 사용 기록
  • 사용 데이터: 앱 내 활동 기록
  • 모든 데이터는 Firebase를 통해 암호화되어 저장됩니다

2. 개인정보의 처리 목적

  • 회원 관리 및 본인 확인
  • 데이터 저장 및 기기 간 동기화
  • 앱 사용 통계 분석(Firebase Analytics)
  • 서비스 개선을 위한 사용자 피드백 수집

3. 개인정보의 제3자 제공

  • Firebase (Google)
    제공 항목: 이메일, 사용 데이터, 디바이스 정보
    제공 목적: 데이터 저장, 동기화, 통계 분석
    보유 기간: 회원 탈퇴 시까지

4. 개인정보의 보유 및 파기

  • 회원 탈퇴 시 즉시 삭제 (법령에 의한 보관 의무 제외)
  • 미사용 계정: 1년 미접속 시 별도 저장 후 파기

5. 이용자의 권리

  • 개인정보 열람, 수정, 삭제 요청 가능
  • 앱 내 설정에서 계정 삭제 가능
